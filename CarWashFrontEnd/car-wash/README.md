# Installation
1. <a href="https://github.com/Hanie-Yekta/MobileCarWash?tab=readme-ov-file#installation">**Clone the repository**</a>

<br>

## Project Setup

```sh
cd MobileCarWash
cd CarWashFrontEnd
cd car-wash
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
